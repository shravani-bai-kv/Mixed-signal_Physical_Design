# Macro Pin Access Debug Investigation

## Motivation

During reproduction of the mixed-signal flow, macro pin-access failures were encountered.

## AI Tool

- ChatGPT / Codex

## Purpose

Generate a systematic debugging workflow for identifying and correcting pin-access issues in analog hard macros.

## Areas Investigated

- LEF pin definitions
- Pin geometry
- Pin layer assignment
- Routing accessibility
- Macro obstructions

## Outcome

The generated workflow was used as a structured methodology for diagnosing macro integration issues during the reproduced flow.
