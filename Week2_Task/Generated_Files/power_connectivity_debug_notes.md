# Power Connectivity Debug Investigation

## Motivation

During reproduction of the mixed-signal flow, warnings related to macro power connectivity were encountered.

## AI Tool

- ChatGPT / Codex

## Purpose

Generate a structured workflow for diagnosing power-grid connectivity issues involving analog hard macros.

## Areas Investigated

- FP_PDN_MACRO_HOOKS
- VPWR/VGND naming consistency
- LEF power pin definitions
- PDN generation
- Macro integration

## Outcome

The generated workflow provided a systematic method for identifying and resolving power connectivity issues during mixed-signal implementation.
