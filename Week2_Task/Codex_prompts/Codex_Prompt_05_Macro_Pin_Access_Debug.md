# Codex Prompt 05 - Macro Pin Access Debug

## Objective

Debug OpenROAD/OpenLane macro pin-access failures during mixed-signal implementation.

## Prompt

OpenROAD reports a macro pin-access failure for an analog hard macro.

Tasks:

1. Identify possible causes.
2. Check LEF pin definitions.
3. Check pin geometry and metal layers.
4. Check routing accessibility.
5. Check obstruction definitions.
6. Suggest modifications to improve routability.
7. Provide a step-by-step debugging workflow.

Output only:

- Root-cause checklist
- LEF inspection checklist
- Debug workflow
- Recommended fixes
